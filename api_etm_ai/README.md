# API ETM — импорт каталога etm.ru

Импорт каталога товаров ETM в инфоблок 41 сайта lvtec.ru.

## Настройка

- **config.php** — базовая конфигурация (URL API, логин/пароль, инфоблок, типы цен, склады, лимиты).
- **config_local.php** — переопределение (создайте из `config_local.php.example`), не коммитить.

## API и лимиты

- **Авторизация** `POST /user/login` — 1 запрос в 2 минуты. Ключ сессии хранится в `logs/api_etm_state.json`, действует 8 часов.
- **Характеристики** `GET /goods/{id}` — 1 запрос/сек.
- **Цены** `GET /goods/{id}/price` (или батч до 50 id) — 1 запрос/сек.
- **Остатки** `GET /goods/{id}/remains` — 1 запрос/сек.

## Файлы маппинга

- **columns.csv** — таблица соответствия «название колонки из API» → «код свойства» в инфоблоке.  
  Формат: `Название;CODE` (заголовок), далее строки `русское название;КОД_СВОЙСТВА`.  
  Если колонка из API совпадает с названием в файле, используется указанный код свойства.

- **new_properties.csv** — сюда дописываются **новые** свойства при первом появлении неизвестной колонки.  
  Формат строки: `Название [KOD]` (русское название и код транслитом в одну строку).

## Скрипты

- **index.php** — статистика и логи обмена, просмотр в браузере. Запуск импорта (форма с лимитом).
- **test_format.php** — тест формата данных API: сырой GET «/» и POST `items_data_get`. Просмотр структуры ответа.
- **import.php** — импорт: получение каталога, маппинг свойств, создание/обновление элементов, цены (розничная), остатки, фото, свойство ETM код.

## Импорт

- Товары ищутся по свойству **ETM код** (или создаются).
- Цены пишутся в розничную (тип цен BASE, ID 1).
- Остатки — в «доступное количество» (склад из конфига).
- Фото загружаются по URL из API в `/upload/etm/`.

## Логи и состояние

- В **logs/** пишутся файлы обмена (`exchange_*.log`), тестов (`test_format_*.log`), ошибок (`api_etm_errors.log`).
- **logs/api_etm_state.json** — ключ сессии и метки последних запросов (для соблюдения лимитов). Не коммитить.
- Последняя статистика импорта — `logs/last_import_stats.json`, отображается на **index.php** после редиректа.

## Доступ к API

Базовый URL и учётные данные в config (по умолчанию тестовый сервер `https://itest2.etm.ru/api/v1`).

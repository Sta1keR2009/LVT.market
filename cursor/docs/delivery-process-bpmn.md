# BPMN: процесс доставки заказа с консолидацией в Лыткарино

Итоговый срок доставки = **макс(сроки поставки на склад Лыткарино)** + **срок доставки ТК** до города получателя.

## Диаграмма (Mermaid)

```mermaid
flowchart TB
    subgraph pool_customer["Покупатель"]
        start([Заказ оформлен])
        end_event([Заказ получен])
    end

    subgraph pool_system["Система / Склады-поставщики"]
        gateway_fork{{"Разделение заказа по складам"}}
        task_w1[Поставка позиций склада 1 в Лыткарино]
        task_w2[Поставка позиций склада 2 в Лыткарино]
        task_wn[Поставка позиций склада N в Лыткарино]
        gateway_join{{"Все позиции прибыли на склад Лыткарино"}}
    end

    subgraph pool_lytkarino["Склад Лыткарино"]
        task_assembly[Сборка заказа]
        task_handover[Передача отправления ТК]
    end

    subgraph pool_tk["Транспортная компания"]
        task_delivery[Доставка до города получателя]
    end

    start --> gateway_fork
    gateway_fork --> task_w1
    gateway_fork --> task_w2
    gateway_fork --> task_wn
    task_w1 --> gateway_join
    task_w2 --> gateway_join
    task_wn --> gateway_join
    gateway_join --> task_assembly
    task_assembly --> task_handover
    task_handover --> task_delivery
    task_delivery --> end_event
```

## Упрощённый вид (один поток по срокам)

```mermaid
flowchart LR
    A([Заказ]) --> B[Поставки с складов 1..N в Лыткарино]
    B --> C["Ожидание сборки заказа (max сроков)"]
    C --> D[Отправка ТК из Лыткарино]
    D --> E[Доставка ТК в город получателя]
    E --> F([Получение заказа])
```

## Легенда

| Элемент | Описание |
|--------|----------|
| **Разделение по складам** | Заказ может содержать позиции с разных складов; каждая часть идёт своим сроком поставки на Лыткарино. |
| **Все позиции прибыли** | Параллельное слияние (AND): отправка возможна только когда все части заказа на складе в Лыткарино. |
| **Сборка заказа** | Формирование единой отправки на складе Лыткарино. |
| **Доставка ТК** | Этап, который показывает калькулятор в карточке товара: тарифы и сроки от Лыткарино до города получателя (DPD, СДЭК и др.). |

Файл BPMN 2.0 (XML) для импорта в Camunda Modeler / bpmn.io: `delivery-process.bpmn`.

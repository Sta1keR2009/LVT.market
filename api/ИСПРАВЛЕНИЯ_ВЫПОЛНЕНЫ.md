## 📋 Проверка работоспособности

После исправлений проверьте:

1. **Скрипты импорта работают из CLI:**
   ```bash
   php /var/www/www-root/data/www/lvtgroup.ru/api/load.php
   php /var/www/www-root/data/www/lvtgroup.ru/api/update_price.php
   ```

2. **Скрипты недоступны через HTTP:**
   ```bash
   curl https://lvtgroup.ru/api/load.php
   # Должно вернуть 403 или "Доступ запрещен"
   ```

3. **Скрипты изменения данных требуют авторизацию:**
   - Откройте в браузере без авторизации - должно быть "Доступ запрещен"
   - После авторизации администратора - должны работать

4. **Конфигурационный файл защищен:**
   ```bash
   curl https://lvtgroup.ru/config/api_promelec.php
   # Должно вернуть 403 или пустой ответ
   ```

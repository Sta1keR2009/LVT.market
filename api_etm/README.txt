========================================================================
  ETM iPRO ORDER API - ОТДЕЛЬНАЯ ИНТЕГРАЦИЯ
========================================================================

Эта папка содержит отдельную интеграцию для работы с ETM iPRO Order API.

========================================================================
СТРУКТУРА ПАПКИ:
========================================================================

/api_etm/
├── classes/
│   └── ETMOrderAPI.php         - Класс для работы с ETM iPRO Order API
├── logs/
│   ├── etm_order_api_*.log     - Логи класса ETMOrderAPI
│   └── etm_order_integration_*.log - Логи интеграции с Bitrix
└── README.txt                  - Этот файл

========================================================================
ОТЛИЧИЕ ОТ /api/:
========================================================================

/api/ - Существующая интеграция для импорта товаров (PromElect API)
      - load.php
      - update_price.php
      - НЕ ТРОГАТЬ!

/api_etm/ - Новая интеграция для отправки заказов (ETM iPRO Order API)
          - ETMOrderAPI.php
          - Работает через Bitrix события OnSaleOrderSaved
          - ОТДЕЛЬНАЯ ОТ /api/

========================================================================
КАК ЭТО РАБОТАЕТ:
========================================================================

1. Пользователь создает заказ в Bitrix
   ↓
2. Срабатывает событие OnSaleOrderSaved
   ↓
3. OrderETMOrderHandler (/local/php_interface/classes/)
   ↓
4. Подключает ETMOrderAPI из /api_etm/classes/
   ↓
5. Отправляет заказ в ETM iPRO API
   ↓
6. Логирует результат в /api_etm/logs/
